F4a — Layouts del bloque

El spec menciona 3col / 4col / lista. Mirando Home + Noticias, también veo uso a 2col. ¿Qué layouts soporta F4a?

2 columnas

Cards más grandes (image arriba, todo el texto debajo). Ideal para destacar pocos items en una sección.

3 columnas (default)

El más usado en Home. Equilibrio entre densidad y legibilidad.

4 columnas

Cards compactas, más densidad. Útil para listados de muchas piezas.

List (apilado)

Image pequeña a la izquierda + texto a la derecha, una fila por item. Para sidebars o secciones secundarias.

Recomendación

F4a soporta 3col (default) + 4col + list — los 3 del spec original. 2col se puede añadir más adelante si Home u otro Figma lo pide explícitamente.

Opciones

A

3col + 4col + list (recomendado)

Cumple el spec. Ya cubre Home y bloques en landings.

B

2col + 3col + 4col + list

Más flexibilidad desde día 1.

C

Solo 3col

Mínimo viable. Añadir el resto cuando algún Figma lo pida.
